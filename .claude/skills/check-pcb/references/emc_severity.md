# EMC 严重度处置

| 严重度 | 处置 |
|---|---|
| CRITICAL | fab 前必修（信号穿地平面缝几乎肯定挂）|
| HIGH | 强烈建议修 |
| MEDIUM | 看上下文评估 |
| LOW / INFO | 实验前参考 |

EMC 不是合规预测器 —— 只能提示常见错（覆盖 ~70%），过没过最终看实验室。

详细规则全文 → `references/pcb-emc-rules.md`。
标准对照（FCC / CISPR / 汽车 / 军规）→ `references/emc-standards.md`。
方法论 → `references/emc-methodology.md`。
