# 跨项目电气铁律（迁自工作区 CLAUDE.md）

适用所有 PCB 项目，circuit-design / draw-schematic / draw-pcb 都按这套约束工作。

## 电源域

- **隔离放大器** $V_{DD1}$ ≤ 5.5V → **禁止接 20V 栅驱**
- **传感器 HV 侧电源**：独立隔离 DC-DC（≥1500VDC 隔离）
- **传感器 LV 侧** $V_{DD2}$ = **3.3V**（跟 ADC VREFHI 对齐，不能用 5V）
- **HV 地 ↔ LV 地**：完全不连，靠隔离放大器跨过

## 差分信号链

```
隔离放大器差分输出 → PCB 差分对 → MCU 差分 ADC
全程差分，不要中间转单端
差分 RC 滤波必须用跨差分对的单电容（C_diff），不要每路对地接电容
```

## 控制平台默认（如果项目 CLAUDE.md 未指定）

由 `USER.md` §"在手硬件" 决定 —— 读它拿具体型号，以及该板的 ADC 位数 / 通道数、PWM 资源、默认开关频率、引脚约定（如 LaunchPad 的 J1–J8）。这些是 user 数据，不在本文件重复。

**注**：MCU 不是技术最优，是 USER.md 决定的。新项目可以在项目 CLAUDE.md 覆盖。
